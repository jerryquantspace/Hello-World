# Hello-World
This is a simple Python program that demonstrates a typewriter-style text animation. It prints messages character by character with a small delay, creating a dynamic, animated effect.

Features:

- Waits for user input before starting.
- Displays text with a smooth typing animation.
- Prompts the user before closing, keeping the final message on screen.
- Easy to customize with your own messages and typing speed.
- Perfect for beginners learning Python, experimenting with loops, or creating small interactive text-based projects.
